Logs messages to Mongo and catches links, for irc.freenode.net, #dailyjs.

h3. Requirements

* "Mongoose":http://github.com/LearnBoost/mongoose
* "Node IRC":http://github.com/martynsmith/node-irc

h3. Installation

Copy settings.js.example to settings.js and edit the settings.
